Bienvenido a Celiapp


CeliAPP es una aplicación Web la cual es nuestra manera de ayudar a los celíacos y a su entorno a facilitar el dilema del día a día respondiendo a la preguenta:

¿Que puedo comer?

A través de nuestra página ofrecemos una manera sencilla y cómoda de buscar los productos aptos para celíacos y obtener información sobre la enfermedad, para el entorno de quienes padecen ésta condición. A partir de una lista de productos brindados por el ente ANMAT podrás buscar de forma rápida y sencilla productos actos para el consumo celiaco, además posee una lista de precio sugeridos con las ubicaciones de cada uno.
